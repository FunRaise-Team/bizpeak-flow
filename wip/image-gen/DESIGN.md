# DESIGN.md — BizPeak Flow 提案配圖（FUNRAISE 品牌、印刷級工程美學）

This is a brand-constrained editorial set: four distinct pieces for a business proposal page. The design system prescribes palette + typography discipline, NOT layout — pick the composition that serves each piece.

## Brand palette (hard)
- Base: white paper #FFFFFF（大面積留白）
- Ink: #333333（主要線條與文字）、#969B9B（次要）
- Accent: FUNRAISE yellow #FAC800 — 單一強調色、克制使用（一張圖內 5–15% 面積）
- 禁：紫藍漸層、glossy 3D、霓虹、glassmorphism、彩虹多色

## Mood vocabulary（借鏡、非模仿）
- 瑞士排版 / 工程製圖（technical drawing、hairline、標號系統）
- 日本說明書美學（分解圖、編號、細節註記）
- The Pudding / 印刷年報的資訊圖表傳統
- 「一眼是真人設計師打磨」— 非 AI 生成俗套（禁置中大標語 + 卡片陣列感）

## Typography
- 中文為主（繁體、可辨識、字要正確）、英文僅產品名 BizPeak Flow 與代號（C0-C7 等）
- 中文字寧大寧少 — 小字群容易崩、避免密集小字段落

## Hard rules (only constraints)
- Generation: image_gen single call per piece, native size, no post-processing
- Content: no emoji / QR / URL / hashtag / watermark / 「Q」 / version 字樣
- Quality: Pentagram / Nikkei Design / 瑞士海報級
